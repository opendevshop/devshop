 # DevShop Docker Images

## `devshop/core` 

A standard base image and Ansible play for all other servers to be built on.

Includes:

- `devshop/base`
    - Operating System
    - SystemD & SystemD in Docker tools.
- `devshop/ansible`
    - Python3, PIP & Ansible
